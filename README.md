## Overview
This is a desktop Solitaire game built using C# and WPF.  
The application includes gameplay features, user accounts, and a scoring system.

## Features
- Login / Sign Up system
- Guest mode
- Difficulty levels (Easy / Medium)
- Vegas Mode (with balance system)
- Timer, score, and move tracking
- Statistics tracking

## Technologies Used
- C#
- WPF (Windows Presentation Foundation)
- CSV file for player saving

## How to Run
1. Download the release from GitHub
2. Run the executable file
3. Login or play as guest

## Version
v1.0 – Initial release

## Future Improvements
- Drag and drop card movement
- Double-click to move cards
- Rules/help screen
- Cash-out system for Vegas mode

## Notes
